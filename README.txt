AI Powered Smart File Organizer

This project was developed as part of the Artificial Intelligence course project (INF340)

About the Project
It is a system that analyzes complex files (PDF, DOCX, JPG, MP3, etc.) on the desktop.
1. Rule-Based: Quickly separates formats such as music and images.
2. Artificial Intelligence: Analyzes ambiguous file names semantically and categorizes them (School, Finance, Project, etc.).
